## 交易感知的新未来

Ergodika 是一个桌面优先的金融驾驶舱，面向实时市场感知。
它通过清晰的可视化与音频辅助信号，减少信息噪音，帮助你更快理解市场结构。

## 最终用户可获得的能力

- 实时加密现货市场监控
- 订单流分析（CVD、OBI、VWAP Anchor）
- 市场动能与活动信号（Kinetic Impact）
- Quartet 四资产驾驶舱布局
- 轻量图表与聚焦诊断信息
- 桌面端音频 MVP 被动监控

## 为什么使用 Ergodika

- 降低长时间交易中的认知负担
- 更早识别市场状态切换
- 在单一驾驶舱中保持纪律化监控流程

## 账户与安全模型

- 交易所账户连接为可选
- 凭据在本地加密存储
- 架构上采用 read-only 监控模式

请仅使用你自己的账户，并遵守当地法律法规。

## 快速开始

1. 克隆本仓库。
2. 安装依赖并启动应用。
3. 建议优先使用桌面模式以获得完整体验。
4. 配置你关注的交易对与布局。
5. 如有需要，连接交易所账户以启用账户相关指标。
