## 在人群反应前，先听见市场

Ergodika 将混乱的市场数据转化为实时的可视化与声音 intelligence。
先听见压力变化，更快读懂结构，在秒级窗口中更果断地行动。

## 最终用户可获得的能力

- 实时加密现货市场监控
- 订单流分析（CVD、OBI、VWAP Anchor）
- 市场动能与活动信号（Kinetic Impact）
- **Scalping score**（0–100）：V-Matrix 中的紧凑 pre-entry 执行质量读数
- Quartet 四资产驾驶舱布局
- 轻量图表与聚焦诊断信息
- 桌面端音频 MVP 被动监控

## 为什么使用 Ergodika

- 降低长时间交易中的认知负担
- 更早识别市场状态切换
- 在单一驾驶舱中保持纪律化监控流程

## 账户与安全

- 架构上采用 read-only 监控模式
- 使用 cockpit 无需连接交易所账户

## 快速开始

1. 克隆本仓库。
2. 安装依赖并启动应用。
3. 建议优先使用桌面模式以获得完整体验。
4. 配置你关注的交易对与布局。
