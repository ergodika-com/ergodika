## Услышь рынок раньше толпы

Ergodika превращает хаотичные рыночные данные в визуальную и звуковую intelligence в реальном времени.
Слышите сдвиги давления, быстрее читаете структуру и действуете уверенно, когда важна каждая секунда.

## Что получает конечный пользователь

- Мониторинг crypto spot в реальном времени
- Order-flow аналитика (CVD, OBI, VWAP Anchor)
- Сигналы рыночной динамики (Kinetic Impact)
- **Scalping score** (0–100): компактная оценка качества исполнения перед входом в V-Matrix
- Макет Quartet для одновременного контроля 4 активов
- Легкие графики и сфокусированная диагностика
- Desktop audio MVP для пассивного мониторинга

## Зачем использовать Ergodika

- Снижение когнитивной нагрузки в длинных сессиях
- Более раннее обнаружение смены рыночного режима
- Единый дисциплинированный рабочий cockpit

## Безопасность

- Архитектурный режим read-only для мониторинга
- Подключение биржевого аккаунта не требуется для работы cockpit

## Быстрый старт

1. Клонируйте этот репозиторий.
2. Установите зависимости и запустите приложение.
3. Для полного опыта используйте desktop-режим.
4. Настройте символы и layout.
