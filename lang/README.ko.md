## 군중보다 먼저 시장을 들어라

Ergodika는 혼잡한 시장 데이터를 실시간 시각 및 사운드 intelligence로 변환합니다.
압력 변화를 먼저 듣고, 구조를 더 빠르게 읽어, 초 단위의 순간에 더 단호하게 대응할 수 있습니다.

## 최종 사용자를 위한 핵심 기능

- 실시간 크립토 현물 시장 모니터링
- 오더플로우 분석(CVD, OBI, VWAP Anchor)
- 시장 활동 신호(Kinetic Impact)
- **Scalping score**(0–100): V-Matrix의 compact pre-entry 실행 품질 리드아웃
- 4개 자산 동시 관측용 Quartet 레이아웃
- 경량 차트와 집중형 진단 정보
- 데스크톱 오디오 MVP 기반 수동 모니터링

## Ergodika를 쓰는 이유

- 장시간 세션에서 인지 과부하 완화
- 시장 레짐 전환을 더 빠르게 감지
- 하나의 콕핏에서 규율 있는 모니터링 워크플로우 유지

## 보안

- 아키텍처상 read-only 모니터링 모델
- cockpit 사용에 거래소 계정 연결 불필요

## 빠른 시작

1. 이 저장소를 클론합니다.
2. 의존성을 설치하고 앱을 실행합니다.
3. 전체 기능은 데스크톱 모드 사용을 권장합니다.
4. 심볼과 레이아웃을 구성합니다.

## 리스크 고지

Ergodika는 투자 자문이 아닙니다.
모든 트레이딩 결정과 결과의 책임은 사용자에게 있습니다.
