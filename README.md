## English

### What is Ergodika

Ergodika is a desktop-first financial cockpit focused on real-time market reading and decision support.
It combines charting, market flow indicators, and guided visual feedback in one interface.

### Main features for end users

- Real-time market dashboard
- Order-flow indicators (CVD, OBI, VWAP Anchor)
- Visual cockpit layout for quick interpretation
- Desktop-first experience (with web mode where available)
- Optional exchange account connection

### Who it is for

- Traders and analysts who want structured market context
- Users who prefer a visual decision cockpit over raw terminal data
- Users who need a practical workflow, not only charts

### Getting started

1. Download or clone this repository.
2. Follow setup instructions in this repo (when published).
3. Launch desktop mode (recommended) or web mode.
4. Configure your preferred symbols and layout.
5. Optionally connect your exchange account.

### Account and data

- Exchange connection is optional.
- If account integration is enabled, credentials must be stored securely.
- Use only your own account and comply with your local laws.

### Risk notice

Ergodika is an analysis tool, not financial advice.
Trading involves risk, including full capital loss.
You are fully responsible for your decisions.

### Privacy and security

- Keep your device secure.
- Never share your credentials.
- Revoke API keys immediately if you suspect compromise.

### Support

For updates, documentation, and support channels, check this repository regularly.
